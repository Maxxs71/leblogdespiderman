# Projet Le blog de Spiderman 

### Cloner le projet 

````
git clone : https://github.com/Maxxs71/leblogdespiderman.git
````
### Deplacer le terminal dans le dossier cloné 
````
cd leblogdespiderman
````

### Installer les vendors (pour recréer le dossier vendor)

````
composer install
````

### Lancer le serveur
````
symfony serve
````
